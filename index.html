What is ELIO

One Board.
Every Connection.

ELIO is a Nordic nRF52-based physical computing platform designed for makers, educators, and AI developers. Connect motors, sensors, and cameras — then control everything from blocks, Python, Arduino, or an AI agent.

// ELIO Board — Nordic nRF52 SDK example
// Flash via neoFlash SWD in < 3 seconds

#include "nrf_gpio.h"
#include "ble_nus.h"
#include "elio_motor.h"

// ELIO port definitions
#define ELIO_DC1_PWM    NRF_GPIO_PIN_MAP(0, 26)
#define ELIO_DC2_PWM    NRF_GPIO_PIN_MAP(0, 27)
#define ELIO_BLE_NAME   "ELIO_DEVICE"

void ble_data_handler(uint8_t *data, uint16_t len) {
    // Parse ELIO 15-byte command packet
    if (data[0] == STX && data[len-1] == ETX) {
        elio_motor_set(data[3], data[4]);  // DC1, DC2
        elio_servo_set(data[5], data[6]);  // SV1, SV2
    }
}

What is MUSA?

MUSA (MBD-1000/BT) is Mobilian's wireless extension module built on the same Nordic nRF52840 chip as ELIO. It bridges Bluetooth 5 Long Range, LoRa, and RS-485 — turning ELIO into a long-distance control system for robotics, smart city infrastructure, and industrial IoT.

For Scratch, Entry, Python & AI Agent on PC

nRF52840 BLE Dongle

When coding from a PC without built-in Bluetooth, plug in the nRF52840 USB Dongle — it acts as a BLE host so Scratch, Entry, and Python can communicate with ELIO wirelessly.

from elio import Elio

board = Elio()
board.connect()           # Bluetooth connect
board.dc(1, speed=80)     # DC Motor 1 → 80% forward
board.servo(1, angle=45)  # Servo 1 → 45 degrees
board.io(1, value=100)    # IO Port 1 → LED full bright
#include <Elio.h>

Elio elio;

void setup() {
  elio.begin();
}

void loop() {
  elio.dc(1, 80);      // DC Motor 1: 80% speed
  elio.servo(1, 45);   // Servo 1: 45 degrees
  delay(1000);
  elio.stop();
}
